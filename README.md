# wrepl

A leiningen plugin to start a repl which responds to http post requests. 

## Usage
### Starting wrepl
lein wrepl [port]

### Connecting to wrepl
you can evaluate clojure expressions by POSTING the expression to http://localhost:port/wrepl/eval
